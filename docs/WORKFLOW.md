# Workflow

## Loop
1. Read `AGENTS.md`.
2. Read `docs/AGENT_HANDOFF.md`.
3. Read `docs/DESIGN_AND_STANDARDS.md`.
4. Read active items in `workstreams/*_todo.yaml`.
5. Implement one focused slice.
6. Update todo status and notes.
7. Commit at user-confirmed healthy feedback points.

## Rules
- Keep changes scoped to one slice.
- Keep docs aligned with code in the same slice.
- Keep progress details in todo files.
